Mes Projets

Restafy

Restafy est une application de gestion de restaurants que j'ai développée pour aider les propriétaires de restaurants à gérer efficacement leurs opérations. L'application offre une interface conviviale et intuitive, avec des fonctionnalités telles que la gestion des réservations, la gestion des menus, la gestion des commandes et la gestion des employés.

  • HTML
  • CSS
  • JavaScript

GetShit

GetShit est une application de gestion de tâches que j'ai développée pour aider les utilisateurs à organiser et à suivre leurs tâches quotidiennes. L'application offre une interface conviviale et intuitive, avec des fonctionnalités telles que la création de listes de tâches, la définition de rappels, la catégorisation des tâches et la possibilité de partager des listes avec d'autres utilisateurs.

  • HTML
  • CSS grid
  • JavaScript

WeatherApp

WeatherApp est une application de prévisions météorologiques que j'ai développée pour fournir aux utilisateurs des informations précises sur les conditions météorologiques actuelles et à venir. L'application utilise des API de données météorologiques pour afficher les prévisions, les températures, les conditions météorologiques et d'autres informations pertinentes pour aider les utilisateurs à planifier leurs activités en fonction du temps.

  • HTML
  • CSS
  • JavaScript
